# Bergmann

Менеджер паролей с терминальным интерфейсом и шифрованием по ГОСТ.

## Demo

https://github.com/wstswsb/bergmann/assets/61021258/10adbe72-519a-485c-832a-6237b1978c6c

## Возможности

- Хранение записей (логин, пароль, URL, заметка)
- Шифрование базы данных алгоритмом Grasshopper (ГОСТ Р 34.12-2015)
- Копирование пароля в буфер обмена
- Полностью терминальный интерфейс (TUI) на базе Textual

## Требования

- Python 3.12+
- [Poetry](https://python-poetry.org/)

## Установка

```bash
pip install bergmann
```

Или через Poetry:

```bash
poetry add bergmann
```

## Использование

```bash
bergmann
```

При первом запуске будет предложено создать новую базу данных и задать мастер-пароль. При повторном — ввести мастер-пароль для расшифровки существующей базы.
